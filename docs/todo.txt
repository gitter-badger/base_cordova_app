 Админка.
 страница содержит три вкладки (тесты, картинки, результаты).

Вкладка “тесты”. Cодержит список существующих тестов, на каждом тесте имеется чекбокс, который определяет тот тест, который будет отображаться.
Также на ней находится форма с полем ввода названия теста, полем ввода времени на тест, описанием теста  и кнопкой “Создать новый тест”. При нажатии на эту кнопку в списке добавляется новый тест с названием введенным в поле название теста и описанием теста.
При нажати на тест открывается форма содержащая список вопросов с кнопками “удалить”, чекбоксами и кнопка “Добавить вопрос”.При выборе чекбокса вопросс будет отображатся в тесте. При нажатии на вопрос или на кнопку добавить вопрос открывается страница “редактирования вопросов”.
На странице “редактирование вопросов” содержится редактор вопроса(текст вопроса тип вопроса, список вариантов ответов с чекбоксами выбора правильных ответов, выпадающим списком изображений).

Вкладка “картинки” содержит список картинок, залитых на сервер и загрузчик картинок.

Вкладка “результаты” содержит список результатов тестирования с фильтрами. список содержит имя, email, ip,балы прохождения тестов, время прохождения, дату и время заполнения.

Клиент.
На главной странице форма ввода имени и email и кнопка “начать тест”.
Страницы с вопросами, содержат вопросы, таймер, кнопку “далее” при нажатии на которую, имеется возможность пропустить вопрос, который будет показан в конце.
При нажатии кнопки “далее”, производится синхронизация с сервером.
Страница результата отображает время и результат

Сервер.
регистрирует пользователя формирует токен для сессии при завершении теста токен изменяется, также он изменяется по прошествии времени выделенном на тест.По мимо токена производится проверка по email.
